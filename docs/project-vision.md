# Project Vision (Template)

Use this file at the start of a project to capture the idea, intended direction, and the main things worth validating first.

This is a planning document, not a contract. It is expected to be incomplete, speculative, and partly wrong early on.

After bootstrap, copy only the few most decision-relevant points from this file into `AGENTS.md` under `Active Project Context (Tier 1 Memory)`.

How to use it:

1. Fill what you know.
2. Leave unknown sections empty or mark them `TBD`.
3. Delete sections that do not matter for the project.
4. Prefer rough clarity over polished prose.
